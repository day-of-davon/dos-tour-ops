Per-user scoping

Auth
Supabase JWT (session.access_token) → resolves user.id
Gmail
session.provider_token = caller's own Google OAuth · queries that user's mailbox only
Cache key
intel_cache(user_id, show_id) · TTL 60 min
Special show_ids
__bulk__ · __label_scan · {venue}__{date}
Sharing
opt-in is_shared flag · readers see sharedByOthers[] from peers on same show
Stale token
401/403 from Gmail → 402 gmail_token_expired → frontend prompts re-sign-in

Classification taxonomy

Categories

please sign · awaiting · following up · bumping · deadline · urgent · ASAP · time-sensitive

Flight dedup
PNR (6-char alnum) preferred → fallback name + date + route
Show match
venue (10) · promoter words (3) · city (4) · country (1) · date proximity (1–3)
Threshold
min score ≥ 4 to attach a thread to a show
