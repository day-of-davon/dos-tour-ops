# tRPC Stack Status

_Last consolidated: 2026-07-19_

## Current Status

- tRPC v11.16.0 (March 28, 2026) and v11.18.0 (June 18, 2026) released.
- No breaking changes from v11.0. DOS Platform tRPC dependency is current and stable.
- v11.18 adds OpenAPI server URL support.

## Ecosystem Watch

- oRPC is emerging as a lightweight tRPC alternative in 2026 SaaS boilerplate comparisons.
- DOS Platform is tRPC-committed; monitor as the ecosystem fragments.
